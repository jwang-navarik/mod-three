# mod-three
This repo implements the Modulo Three Exercise. There are 3 files in src.

## src/fsm.js
This is the Finite State Machine

## src/mod-three.js
This is the main function that does modulo three calculation. It calls fsm as required.

## src/gen-fsm.js
This is FSM generator for the Advanced Exercise

## \_\_tests__
These are the tests. To run tests, just go with "npm run test" or "jest".
